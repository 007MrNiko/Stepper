(розроблена спеціально для роботи на льоту) + не увсіх телефонах працює JavaScript

При вході на сайт, можливо знадобиться певний час. Через те що він хоститься на сервісі Heroku, то при довгій неактивності вимикається
та для реактивації потрібен певний час. Проте після цього все буде нормально
(це від нас не залежить питання до сайту Heroku і можливостей безплатної версії https://devcenter.heroku.com/articles/how-heroku-works)
